Fusion Starter
Setup Guide

The quickest clean setup for Fusion. Start with AIOStreams, add Torbox if you use debrid, then use the Starter Kit to make the homescreen, ratings, overlays, and trailer previews feel complete.

01

Start Here

This guide keeps the first setup intentionally simple. Fusion plus AIOStreams is enough for a strong basic build. If you already have a sources addon or AIOStreams configured, you can skip that part and use your existing setup instead.

Already have sources set up?
If yes→Skip the addon setup part

If you already have a working sources addon or AIOStreams configuration, use that directly and skip ahead to the Starter Kit, ratings icons, poster overlays, or trailer previews.

Keep the stack minimal
Use→Fusion + AIOStreams + Torbox

AIOStreams is the recommended addon. If you use debrid, pair it with Torbox. This guide does not include AIOMetadata as part of the default setup.

Use the community resources
Add later→Starter Kit, BetterPosters, Trailerio

Most of the best Fusion polish already exists. Use the Fusion Starter Kit for homescreen setups, widgets, collections, and icon packs. Then use BetterPosters and Trailerio when you want the interface to feel finished.

Playback picks

Start with Fusion's internal player. Move to Infuse only if you want the premium Apple-device playback path.

02

AIOStreams

For a basic Fusion setup, this is the addon to start with. Use the official AIOStreams docs for the current flow. If you already have AIOStreams or another sources addon configured, you can skip this section and use that instead.

Open the official docs
Use→docs.aiostreams.viren070.me
OpenAIOStreams docs

Use the official AIOStreams documentation to configure the addon and generate the install URL you want to use in Fusion. If you use debrid, Torbox is the recommended provider alongside AIOStreams.

Install the generated addon URL
Paste into→Fusion Addons
Find itSettings›Addons

Once the docs give you the final AIOStreams URL, paste it into Fusion and let the app load the addon manifest.

03

Starter Kit

The Fusion Starter Kit is the easiest way to import homescreen layouts, widgets, collections, icon packs, and other resources.

Open the Starter Kit
Use→Fusion community project
Openitsrenoria.github.io›fusion-starter-kit

Browse the Starter Kit for homescreen setups, widgets, collections, icon packs, and other community-built Fusion resources.

Choose sourced or self-assigned imports
When asked→Pick your source behavior

Many Starter Kit imports ask whether you want sources included or want to assign them yourself in Fusion. Choose sourced for a ready-made setup, or raw if you want to wire rows manually.

The goal is to get Fusion working cleanly with the fewest moving parts first. Add complexity only after the basics feel stable. In practice, widgets and collections are almost the same import flow.

Import widgets Most common
The fastest way to add rows to the home screen.
Find it Settings› Widgets› Scroll to bottom› Import› URL

Pick a widget from the Starter Kit, click Copy JSON URL, choose Widget in Fusion, then decide whether you want sources included or want to assign them yourself later.

Import collections Almost identical
Collections follow nearly the same idea as widgets.
Find it Settings› Widgets› Add New› Collections Row› Continue› Import Collections

Pick a widget from the Starter Kit, click Copy JSON URL, choose Collection, then choose whether you want the sources bundled in or want to attach them yourself in Fusion.

Import full setups Fastest reset
Best when you want Fusion to feel complete in one move.
Find it Settings› Widgets› Scroll to bottom› Import› URL

Pick a full setup in the Starter Kit, click Copy JSON URL, select Full Setup, then paste it into Fusion.

Import BYO setups Build your own
Use JSON paste when you want a custom widget bundle.
Find it Settings› Widgets› Scroll to bottom› Import› Paste JSON

Choose BYO in the Starter Kit, click Build JSON, select the widgets you want, click Copy JSON, choose whether you want sources included, and paste the JSON string into Fusion.

04

Ratings Icons

Fusion can show community rating labels on posters. To make that work cleanly, connect PMDB first, then import an icon pack and map it to your labels.

1

Connect PMDB

Go to Settings › Accounts › PublicMetaDB and connect your account.

2

Enable community ratings

Open Settings › Metadata › Community Ratings and toggle it on.

3

Create the labels

Make labels like masterpiece, great, good, okay, and skip. Reorder them if you want.

4

Import an icon pack

Go to Settings › Icon Packs and import from URL or paste JSON. The Starter Kit includes icon pack links for this.

5

Assign icons to labels

Tap each rating label and choose the icon you want attached to it.

6

Show them on posters

Back in community ratings, make sure Show on Posters is enabled.

PMDB and icon packs
The Fusion Starter Kit is the easiest place to find icon packs. PMDB itself is also worth knowing about beyond ratings: it is an alternative to Trakt, built by the Fusion developer. Official site: publicmetadb.com.

Trailer previews
Install→Trailerio manifest
Use URLtrailerio.cc/manifest.json

Add Trailerio in Fusion so the app can surface trailer previews in the hero area.

Enable trailer previews
tvOS: Open Settings › Video Player and turn Autoplay Previews on. The exact wording can vary slightly by version, but it stays in the video player settings.
